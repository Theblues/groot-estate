## Getting Started

Install the module with: `npm install shapes`

```js
var shapes = require('shapes');
shapes.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g shapes
$ shapes --help
$ shapes --version
```


```sh
# creates a browser.js
$ grunt browserify
```
